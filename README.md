# 🌍 Discover Puerto Rico Data Scraper

[![GitHub Repo](https://img.shields.io/badge/GitHub-Repo-blue?logo=github)](https://github.com/Sacarianos/Discover-Puerto-Rico-data-scraper)

## 🚀 Overview

**Discover Puerto Rico Data Scraper** is a web scraping tool designed to extract structured information from **DiscoverPuertoRico.com**. The tool collects data on **things to do, restaurants, bars, hotels, and accommodations**, making it useful for **travel planning applications**, **recommendation systems**, and **local business insights**.

---

## 🛠 Features
✔ **Extracts structured data from DiscoverPuertoRico.com**  
✔ **Headless web scraping using Selenium & Browserless.io**  
✔ **Handles JavaScript-rendered content**  
✔ **Stores data in structured formats (CSV/JSON/Database)**  
✔ **Optimized for performance & scalability**
